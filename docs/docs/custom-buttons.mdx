# Custom Buttons 

> You can change the Prev and Next arrows, as you like, as shown in the following example

<Example/>

### 1. First create the functions that will wrap the new buttons

```jsx title="exampleButtons.jsx"
function CustomNextButton() {
   return (
        <div className="next-icon"></div>
   )
}

function CustomPrevButton() {
    return (
        <div className="prev-icon"></div>
    )
 }
```

### 2. Then pass the functions to the slider using props

```jsx title="exampleButtons.jsx" {4,5}
export default function ExampleButtons() {
    return (
        <Helios
            nextArrow={{ component: <CustomNextButton /> }}
            prevArrow={{ component: <CustomPrevButton /> }}
        >
            <div className="box">
                <h1>
                    Slider 1
                </h1>
            </div>
            <div className="box">
                <h1>
                    Slider 2
                </h1>
            </div>
            {/** More Elements*/}
        </Helios>
    )
}
```

### 3. Custom CSS
> You can also add custom classes to the buttons, as shown in the following example

<ExampleCSS/>
<br/>

```jsx title="exampleButtons.jsx" {2,3}
<Helios
    nextArrow={{ component: <CustomNextButton />, class: 'your-custom-next-class' }}
    prevArrow={{ component: <CustomPrevButton />, class: 'your-custom-prev-class' }}
>
```

```css title="exampleButtons.css"
.your-custom-next-class {
    position: absolute;
    bottom: 0;
    right: 0;
    cursor: pointer;
}

.your-custom-prev-class {
    position: absolute;
    bottom: 0;
    right: 44px;
    cursor: pointer;
}
```
