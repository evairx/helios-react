# Dots

> Activating this option will display the Dots in your slider.

```html title="index.html" {7}
<!-- Slider Elements -->
<script src="https://cdn.jsdelivr.net/npm/@evairx/heliosjs/helios.min.js"></script>

<script>
    let helios = HeliosConfig({
        infinite: true,
        dots: true,
    });
</script>
```
